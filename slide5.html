Componentes de MySQL

El servidor

Programa principal que corre como servicio en Linux o Windows. Coordina operaciones y gestiona el acceso concurrente garantizando la integridad de datos.

Cliente y utilidades

Programas para interactuar con el servidor MySQL:

MySQL: Consola interactiva
mysqladmin: Administración
mysqldump: Copias de seguridad
mysqlcheck: Optimización

Lenguaje SQL

Implementación del estándar SQL que permite realizar operaciones de:

DDL: Definición de Datos (CREATE, ALTER, DROP)
DML: Manipulación de Datos (SELECT, INSERT, UPDATE)

Directorio de datos

Ubicación física donde se almacenan las bases de datos. Incluye:

Carpetas organizadas por base de datos
Archivos de datos (.MYD) e índices (.MYI)
Archivos de formato (.frm) con definiciones
